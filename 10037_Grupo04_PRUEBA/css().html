jQuery - método css()

El css()método establece o devuelve una o más propiedades de estilo para los elementos seleccionados.

Devolver una propiedad CSS

Para devolver el valor de una propiedad CSS específica, use la siguiente sintaxis:

               
                  css("propertyname");

El siguiente ejemplo devolverá el valor de color de fondo del PRIMER elemento coincidente:

Ejemplo 

               
                  $("p").css("background-color");

Establecer una propiedad CSS

Para establecer una propiedad CSS específica, use la siguiente sintaxis:

         
            css("propertyname","value");

El siguiente ejemplo establecerá el valor de color de fondo para TODOS los elementos coincidentes:

Ejemplo 

         
            $("p").css("background-color");
