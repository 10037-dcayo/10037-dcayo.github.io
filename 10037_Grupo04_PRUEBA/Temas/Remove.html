JQUERY REMOVE

Con jQuery, es fácil eliminar elementos HTML existentes.

Eliminar elementos/contenido

Para eliminar elementos y contenido, existen principalmente dos métodos jQuery:
remove()- Elimina el elemento seleccionado (y sus elementos secundarios)
empty()- Elimina los elementos secundarios del elemento seleccionado

Ejemplo 1


<!DOCTYPE html>
<html>
<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"</script>
<script>
$(document).ready(function(){
   $("button").click(function(){
     $("#div1").remove();
   });
 });
</script>
</body>
<div id="div1" style="height:100px;width:300px;border:1px solid black;background-color:yellow;">
   This is some text in the div.
   <p>This is a paragraph in the div.</p>
   </div>
   <br>
   <button class="btn btn-primary btn-lg text-black">Remove div element</button>

Demostración:

This is some text in the div.

This is a paragraph in the div.


Método jQuery vacío ()

El empty()método jQuery elimina los elementos secundarios de los elementos seleccionados.

Filtrar los Elementos a Eliminar

El método jQuery remove()también acepta un parámetro, que le permite filtrar los elementos que se eliminarán.
El parámetro puede ser cualquiera de las sintaxis del selector de jQuery.
El siguiente ejemplo elimina todos los elementos con class="test":

Ejemplo 2


<html>
<head>
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.6.4/jquery.min.js"</script>
<script>
$(document).ready(function(){
   $("button").click(function(){
     $("p").remove(".test, .demo");
   });
 });
 </script>
 <style>
 .test {
   color: red;
   font-size: 20px;
 }
 .demo {
   color: green;
   font-size: 25px;
 }
</script>
</head>
<p>This is a paragraph.</p>
<p class="test">This is p element with class="test".</p>
<p class="test">This is p element with class="test".</p>
<p class="demo">This is p element with class="demo".</p>
<button class="btn btn-primary btn-lg text-black">Remove all p elements with class="test" and class="demo"</button>
</body>
</html>

Demostración:

This is a paragraph.

This is p element with class="test".

This is p element with class="test".

This is p element with class="demo".
